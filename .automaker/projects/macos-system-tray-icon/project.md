# Project: macOS System Tray Icon

## Goal
Add a persistent macOS system tray icon to the Electron app so Ava stays accessible in the menubar when the main window is closed, with a context menu for quick actions and a live status indicator showing running agent count.

## Milestones
1. macOS System Tray - Add tray icon, window hide behavior, Cmd+Q intercept, and live agent status to the Electron main process.
