# homeMaker documentation

homeMaker is a self-hosted home management hub. It gives you a single place to track house projects, assets, maintenance schedules, budgets, IoT sensor data, and encrypted credentials — with an AI agent to help research and plan.

This documentation covers installing and running homeMaker, using its modules, deploying to production, and integrating with external services.
